matrix multiplication routine, performs C += A*B

Parameters
Mheight of A and C
Nwidth of B and C
Kwidth of A, height of B
Ainput matrix A
incRowAspecifies how to find the next row, this add flexbility to get to the next row, i.e. after taking K entries, there can be a gap between the next row
Binput matrix B
incRowBspecifies how to find the next row in B
Coutput matrix C
incRowCspecifies how to find the next row in C

note that for our purpose, incRowA, incRowB, incRowC may be redundant in cblas, there are also incColA, incColB, incColC, performance in Loki increases a little when eliminating some unnecessary variables
